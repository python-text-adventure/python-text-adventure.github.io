Recursive Sorting Algorithms

Some of the sorting algortihms previously discussed are recursive. It is easiest to see the recursion when you look at the code.

Bubble Sort

"The bubble sort makes multiple passes through a list. It compares adjacent items and exchanges those that are out of order. Each pass through the list places the next largest value in its proper place. In essence, each item 'bubbles' up to the location where it belongs." ("The Bubble Sort", Interactive Python)
Bubble Sort is not recursive.

Insertion Sort

"The insertion sort...works in a slightly different way. It always maintains a sorted sublist in the lower positions of the list. Each new item is then 'inserted' back into the previous sublist such that the sorted sublist is one item larger." ("The Insertion Sort", Interactive Python)
Insertion sort is not recursive.

Merge Sort

Merge sort is a recursive algorithm that continually splits a list in half. If the list is empty or has one item, it is sorted by definition (the base case). If the list has more than one item, we split the list and recursively invoke a merge sort on both halves. Once the two halves are sorted, the fundamental operation, called a merge, is performed. Merging is the process of taking two smaller sorted lists and combining them together into a single, sorted, new list.("The Merge Sort", Interactive Python)

Quick Sort

A quick sort first selects a value, which is called the pivot value. Although there are many different ways to choose the pivot value, we will simply use the first item in the list. The role of the pivot value is to assist with splitting the list. The actual position where the pivot value belongs in the final sorted list, commonly called the split point, will be used to divide the list for subsequent calls to the quick sort.("The Quick Sort", Interactive Python)

Quick Sort is a recursive algorithm.

Sort() and Sorted()

Fortunately for us, now Python has a built-in method for permenantly sorting, sort(), and a built-in function for temporary sorting, sorted().

Using these built-in commands, it is easy for us to sort nested lists.

We can sort other types of nested data using a lambda function, a topic we do not explore in CS 30.
