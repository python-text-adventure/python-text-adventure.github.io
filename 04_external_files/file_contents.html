Reading File Line by Line

If the file or data is too big to read, we can code it so it reads the data line by line by using a method called readlines().

You will notice there is a blank line between each line of digits of pi.

We can put the digits all on the same line by using a method called rstrip() that gets rid of the blank lines.

We can get rid of the whitespacces entirely by using the method strip().
