F-strings

If you recall from previous experience with working with strings, it can be awkward to print the value of variables, especially if they are integers or floats. An easier way is using formatted strings or f-strings. F-strings put the letter f in front of the quotation marks and the variables in curly brackets -- there's no need to convert to a string. F-strings are particularly helpful for text-based adventures because you can reduce the possible user-inputs. For example, if I put all the user-inputs to lower-case letters, it makes my if-elif-else statements much more simple.
