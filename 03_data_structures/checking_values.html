For the small lists, tuples, and dictionaries we've seen so far, it is easy to manually check for data. For large sets of data or when you rely on user input for things like text-based adventure games, you need a way to be able to check for data automatically.

To check a list or tuple for a value, we can use the keyword in to yield a True or False value. We can use these True or False statements to create conditional statements for our RPG. In the code below, the statement

if action_input.lower() in possible_actions:

checks if the user input is in the list of valid actions.
