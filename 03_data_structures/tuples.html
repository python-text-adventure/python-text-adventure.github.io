Tuples

Lesson adapted from Programiz: Python Tuple https://www.programiz.com/python-programming/tuple

As we have seen, lists are ordered and easily changed which means that they are mutable. On the other hand, sometimes you want a data structure that cannot be changed and still ordered, which is immutable. A type of immutable data structure is a tuple. Tuples are best used for static data like GPS coordinates and the key in a dictionary.

Lists vs Tuples

How you can tell the difference between a list and a tuple is the type of brackets used:

  • Lists use square brackets []
  • Tuples use round brackets ()

Tuple Methods

Unlike lists, tuples only have two methods that work on them

  • index() - returns the index of the first occurance of an item
  • count() - counts the number of times an item repeats
