Assignment: RPG Inventory

Create two or more inventories for your text-based adventure game using nested dictionaries or two-dimensional arrays. Each dictionary item must be printed out separately with their descriptions. Additionally, elements of your nested dictionary must be accessible from the working menu and you must print out sentences for each item in each list using f-strings. Add and replace your code in your main game file and push to GitHub.

CS30-FP1: Investigate types of data structures and the advantages of organizing data in different ways.
(b) Develop a program that utilizes a two-dimensional array.
(c) Solve a problem utilizing a mutable array (e.g., list or ArrayList).
(d) Explore the use of associative arrays (e.g., hash tables or dictionaries) to store a dataset.
