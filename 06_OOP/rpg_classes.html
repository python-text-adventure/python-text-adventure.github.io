Task 1: Create Classes

Create classes for elements like inventory, player, obstacles, and map in separate files. Import and incorporate them into the game. At least one class must have a child class.

For the map class and multidimensional arrays:

  • Add direction methods to the main game menu
  • Add inventory and obstacles at various tiles
  • Add a start tile and a victory tile

Task 2: Organize the Classes

Create a sublist using a list comprehension from classes to sort the inventory. For example Items from healing, items for consuming, items for protection. Check if consumables are in the inventory and if the player uses them then the item is removed.

By the end of these two tasks, you will be very close to having a completely functional game.
