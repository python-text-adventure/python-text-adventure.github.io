Libraries

Partially Adapted from Matthes, Eric, "Chapter 9: Classes - The Python Standard Library", Python Crash Course (2 ed), No Starch Press, Inc., 2019

Python is a modular programming language which means that some classes, functions, keywords, and methods are built-in but others are imported when needed. Many modules need to be installed on your computer to work, but the Python Standard Library is a set of modules included with every Python installation. You can use any class of function in the standard library by putting an import statement at the top of your Python file.
