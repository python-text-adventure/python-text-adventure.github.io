CS 20 Review: Introduction to Lists Answers

Practice #1

  1. Names: Store the names of a few of your friends in a list called names. Print each person’s name by accessing each element in the list, one at a time
  2. Greetings: Start with the list you used in question 1, but instead of just printing each person’s name, print a message to them. The text of each message should be the same, but each message should be personalized with the person’s name.
  3. Your Own List: Think of your favorite mode of transportation, such as a motorcycle or a car, and make a list that stores several examples. Use your list to print a series of statements about these items, such as “I would like to own a Honda motorcycle.”

Practice #2

Create a dictionary of at least 5 student names and their grades (these can be made up students). Print the entire dictionary two ways:

  • as a whole
  • Create a sentence about each student by accessing the dictionary

Practice #3

  1. Create a dictionary that stores your first name, last name, and city where you live. Print the entire dictionary.
  2. Add to your dictionary from the name of your school to Campbell Collegiate. Print the updated dictionary.
  3. Change the name of your school. Print the updated dictionary.
  4. Remove your last name from the list and then print the updated dictionary.
