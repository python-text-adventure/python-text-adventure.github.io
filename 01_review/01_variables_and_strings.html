Variable Creation Rules

There's a couple rules that we must follow when we name our variables:

  • variables can only contain letters, numbers, and underscores
  • variables cannot start with a number
  • spaces are not allowed in variable names (you can use underscores in their place)
  • variable names should be short and descriptive. For example: name_length is better than length_of_persons_name
  • To avoid confusion, don't use lowercase l (L), uppercase I (i) and uppercase O (o) as variables. Python will be able to read them, but another use might confuse them.
