CS 20 Review Assignment: RPG Simple Menu

Create a simple menu for your game of possible actions the user can take. After the menu of possible actions are printed, the program must accept user input. A message for each action must be printed to confirm a valid or invalid action was choosen. Make sure to add a loop for continous game play and an option for quitting. You may need to do additional research to complete this assignment.

Outcomes include:
CS30-CP1: Implement effective coding practices throughout Computer Science 30.
CS30-FP1: Investigate types of data structures and the advantages of organizing data in different ways.
